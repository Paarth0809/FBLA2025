FAQs

What inspired your game?

Our curiosity for coding, and our love for our favorite childhood show, Avatar the Last Airbender, inspired this project!

How does your code work?

Our code mainly uses functions, which are pieces of code that can be re-used. We used these as starting blocks for our code.

What element should I choose to play?

You should choose an element based on your playstyle. If you want a character with deep development, choose Fire. If you want to be the main boss of your own story, choose Earth. For a calmer sidekick-style gameplay, go with Water. And if you want the full experience of the show, choose Air to play as the Avatar.
